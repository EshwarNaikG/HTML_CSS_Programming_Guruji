CSS Notes (Basic → Advanced)
1. What is CSS?

CSS (Cascading Style Sheets) is used to style HTML elements.

It controls:
a. Colors
b. Fonts
c. Layouts
d. Animations
e. Responsive design

--------------------------------------------------------------
2. CSS Syntax
selector {
    property: value;
}

Example:

h1 {
    color: blue;
    font-size: 40px;
}

--------------------------------------------------------------------
3. Types of CSS

1. Inline CSS
<h1 style="color:red;">Hello</h1>
------------------------------------------------------------
2. Internal CSS

<style>
h1{
   color: blue;
}
</style>

-------------------------------------------------------------
3. External CSS

<link rel="stylesheet" href="style.css">
---------------------------------------------------
4. CSS Selectors

Element Selector.........................
p{
   color:red;
}

Class Selector......................
.box{
   background:yellow;
}

ID Selector............................
#title{
   color:blue;
}

Universal Selector............................
*{
   margin:0;
}

Group Selector..................................
h1,p,div{
   color:green;
}

35. CSS Best Practices

✅ Use external CSS
✅ Use meaningful class names
✅ Avoid excessive !important
✅ Use Flexbox/Grid
✅ Write responsive code
✅ Keep code clean and reusable

36. Mini Interview Questions
Q1. Difference between ID and Class?
ID	Class
Unique	Reusable
#id	.class
Q2. Difference between Flexbox and Grid?
Flexbox	Grid
1D Layout	2D Layout
Row OR Column	Rows AND Columns
Q3. What is Box Model?

Content + Padding + Border + Margin

Q4. Difference between Relative and Absolute?
Relative → moves from original position
Absolute → moves relative to parent
37. Complete Example
<!DOCTYPE html>
<html>
<head>
<style>

body{
   margin:0;
   font-family:Arial;
}

.container{
   display:flex;
   justify-content:center;
   align-items:center;
   height:100vh;
}

.card{
   width:300px;
   padding:20px;
   border-radius:10px;
   box-shadow:0 0 10px gray;
   transition:0.5s;
}

.card:hover{
   transform:scale(1.05);
}

button{
   padding:10px 20px;
   border:none;
   background:blue;
   color:white;
   cursor:pointer;
}

</style>
</head>

<body>

<div class="container">

<div class="card">
<h1>CSS Card</h1>
<p>Learning CSS is fun.</p>
<button>Click</button>
</div>

</div>

</body>
</html>
